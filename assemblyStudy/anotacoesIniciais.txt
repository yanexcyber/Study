// A arquitetura em 32bits permite a escrita de malwares mais complexos, pois permite o uso de mais memória e registradores, além de instruções mais complexas...

// Máquinas de 64 bits mantém a compatibilidade com as instruções de 32 bits, mas também possuem instruções de 64 bits, que permitem o uso de mais memória e registradores, além de instruções mais complexas...

// A Unidade de Controle direciona o fluxo de dados e instruções dentro do processador, controlando o funcionamento de todos os componentes internos do processador.

// Por mais que a computação avance a arquitetura de Von Neumann ainda é utilizada, pois é a mais simples e eficiente para a maioria das aplicações.

// A memoria guarda apenas bits e não sabe o que eles representam, por isso é necessário que o processador saiba interpretar os bits da memória.

// A memória é dividida em células de memória, que são endereçadas por um endereço de memória, que é um número inteiro que identifica a posição da célula de memória.

// O tamanho de uma célular pode variar de acordo com a arquitetura do processador, mas geralmente é de 8 bits.

// O processador também te memoria interna, chamada de registradores, que são células de memória de alta velocidade, que armazenam dados e instruções que estão sendo processados no momento.
